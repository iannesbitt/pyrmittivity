`pyrmittivity` Documentation
#####################################

pyrmittivity is a simple module designed to calculate Maxwell's Equations
describing Permittivity.

See an explanation of Permittivity at a site dedicated to Maxwell's Equations
`here <https://maxwells-equations.com/materials/permittivity.php>`_.
